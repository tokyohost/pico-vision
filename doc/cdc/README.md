# Third-party USB device modules

`usb/device/core.py` and `usb/device/cdc.py` are vendored without functional
changes from the official `micropython/micropython-lib` repository, commit
`68f8fe4e3c9b408642353c01d9c4f2667399a79b` (2026-07-03).
